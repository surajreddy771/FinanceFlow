# **App Name**: FinanceFlow

## Core Features:

- Expense Input: Allow the user to input expenses.
- Spending Visualization: Visually present user's spending via charts.
- Savings Goals: Let user set goals (e.g., saving for a purchase) with a target amount.
- Personalized Advice: AI-powered tool to create personalised recommendations based on financial data.
- Budget Setting: Allow manual setting of a budget.
- Category Management: Enable category management for incomes and expenses.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke trust and stability.
- Background color: Light blue (#E1F5FE) to create a clean, airy feel.
- Accent color: Yellow-gold (#FFD700) to highlight key financial insights.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
